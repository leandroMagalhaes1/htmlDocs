Os elementos podem ter atributos

Este paragrafo tem um atributo chamado:class com um value:atributo_1

Em HTML existem atributos chamados de atributos booleanos que não precisam de um valor explícito,
    pois a simples presença do atributo indica que ele está ativado ou é verdadeiro. Esses atributos assumem um valor implícito quando presentes,
    e o valor é true. Se estiverem ausentes, o valor é false.

Boolean Attributes

  • checked

    • Este atributo boolean indica que uma caixa de seleção ou botão de radio está marcado

  • disabled

    • Usado para desativar um elemento interativo por exemplo um botão

  • readonly

    • Usado para indicar que o utilizador não pode modificar o valor

  • required

    • Indica que um campo de um formulário deve ser preenchido

  • autofocus

    • Indica que quando a página é carregada este elemento recebe o foco na página

  • multiple

    • Usado num elemento select ou input para permitir itens múltiplos

  • novalidade

    • Usado em formulário para desativar a validação

  • hidden

    • Texto oculto

    Torna um elemento invisivel mas ele ainda existe no DOM

  • formnovalidate

    • Usado em um botão de envio para desativar a validação do formulário quando aquele botão é clicado.

  • async

    • Este atributo funciona só para script elements e quando presente deixa o script correr de forma async

  • autoplay

    • para audio ou para video, quando presente o video toca automaticamente ao dar load

  • controls

    • Existe para fornecer controlos para o audio e video, controlos como: pausa, volume, play

  • default

    • Para track elements para video e audio marca o track como o default

  • defer

    • Para elementos script quando presente força o script a ser executado depois do HTML, mas antes de DOMContentLoaded

  • ismap

    • Para elementos de imagem faz com que qualquer clique que é feito na imagem é mandado as cordenadas para o servidor

  • itemscope
  • loop

    • Faz com que um video ou audio estejam em loop e que nunca parem

  • muted

    • para um video ou audio faz com que quando a pagina da load ele começa sem som

  • nomodule

    • Para elementos script que não devem ser executados em browsers que não usam ES modules

  • novalidate

    • Para formulários quando presente desativa a validação quando enviado

  • open
    • More Information

    Some hidden content

    Para o elemento details indica que dá para clicar e expandir para ver mais

  • reversed
    1. Third Item
    2. Second Item
    3. First Item

    Faz com que uma lista ol conta de trás para a frente

  • selected

    • Este att faz com que em elementos option dentro de elementos select estejam marcados por default

  • truespeed
    • Scrolling text

    faz com que um texto que esteja a passar em loop seja a velocidade verdadeira e nao a default
